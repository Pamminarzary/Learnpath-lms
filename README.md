## ✨ Features

- 🎓 **13 Courses** — Web Dev, Data Science, ML, AI, DevOps, Cloud, Blockchain & more
- 🔍 **Live Search** — Filter courses by name in real time
- 🔐 **Auth System** — Sign up, log in, log out (localStorage-based)
- 📋 **Course Enrollment** — Enroll in any course and track your selections
- 🌙 **Dark Mode** — Persistent theme toggle across all pages
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- ⚡ **No Backend Required** — Pure static site, deploy anywhere

### Run Locally

```bash
cd learnpath-lms

```

No install needed. Just open `landing.html` in any browser.

## ☁️ Deployment

### Deploy to Netlify (Recommended — Free)

1. Go to [netlify.com](https://netlify.com) and sign up
2. Drag and drop the project folder onto the Netlify dashboard
3. Your site is live in under a minute ✅

### Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set Source to: `main` branch → `/ (root)`
4. Your site will be live at `https://your-username.github.io/learnpath-lms`

### Deploy to Vercel

1. Push to GitHub
2. Import repo at [vercel.com](https://vercel.com)
3. Click **Deploy** — done ✅
